# odin-Rock-Paper-Scissors


A simple **Rock Paper Scissors** game written in **JavaScript**, created as part of **The Odin Project – Foundations** curriculum.

This project focuses on core JavaScript concepts such as functions, conditionals, user input, and basic game logic.

---

## Assignment

Create a Rock Paper Scissors game that:
- Runs in the browser
- Uses JavaScript only (no UI required)
- Plays **5 rounds**
- Keeps track of the score
- Outputs results to the console

---

## How It Works

- The game plays **5 rounds**
- Each round:
  - The player enters `rock`, `paper`, or `scissors` using `prompt()`
  - The computer randomly selects a choice
  - The winner of the round is logged to the console
- After 5 rounds, the final score is displayed

---

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice results in a draw
